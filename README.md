# flomo
此方法可在未解锁手机的情况下更快速的输入一段文字并保存到flomo中。

1、下载flomo.js到jsbox中

2、脚本添加到启动器，在负一屏widiget中添加jsbox的启动器

注：iOS14的小组件无法交互，依然使用的ios13的widget

3、点击flomo，发送文字。完成
